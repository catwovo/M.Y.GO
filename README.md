
## 一、 游戏概述

《M.Y.GO》是一款开创性的**卡牌驱动型围棋策略游戏**。游戏打破了传统围棋极高的上手门槛，通过将“卡牌编组”、“锦标赛”与“本地深度学习围棋 AI（KataGo）”以及“大语言模型（LLM）”相结合，让非专业围棋玩家也能在轻松、策略性的氛围中体验职业级的大局观博弈。

---

## 二、 核心玩法机制

### 1. 双模式自由切换

- **卡牌托管模式**：玩家不需要自己寻找落子点。在回合开始时，玩家从自己编好的 20 张卡组中抽取手牌。点击棋手卡牌，本地的 KataGo 引擎会在盘面上推荐 A、B、C 三个最佳候选点。玩家通过阅读右侧的战术点评，权衡利弊后进行落子。
    
- **自主下棋模式**：玩家可以在对局中随时一键开启“自主下棋”。此时卡牌托管将被暂时禁用，玩家可以直接在棋盘任意合法空位上落子，随后 AI 对手将根据其设定的段位和棋风进行回应。

### 2. 真实大局观 AI 研判

- 当打出棋手牌请求推荐时，Godot 前端会实时将当前棋盘的落子状况转化为 **ASCII 字符矩阵** 发送给后台大语言模型（需要自己配置API KEY）。
    
- 大模型在**完全不知道胜率数值**（防止数值偏见）的情况下，纯粹基于棋子物理位置、局部邻居战术关系以及 KataGo 预测的后续变化链（PV），为玩家撰写出高度专业、逻辑严密的纯文字大白话战术分析。
    

---

## 三、 本地深度学习 AI 引擎

游戏在本地集成并运行了先进的 **KataGo 围棋大脑**，并专门引入了官方 **Human SL（人类监督学习）权重模型**：

- **真实人类行棋风格**：该模型基于数千万盘真实人类对局训练而成。AI 的落子不再表现出冰冷、机械的机器感，而是会像人类棋手一样去守角、挂角、飞压或主动发起局部缠斗，极具人情味。
    
- **原生段位与棋风调节**：游戏去除了生硬的随机犯错代码，直接通过 GTP 协议动态控制 KataGo 的 humanSLProfile（段位，支持 20k 至 9d）和 playoutDoublingAdvantage（优势偏差，用于控制力战和稳健棋风）。AI 会在保持高水平棋形的前提下，自然地根据所选级别下出符合该段位水平的缓着或妙手。

## 四、 游戏模式

### 1. 快速游戏

- 玩家可以自定义棋盘路数（支持 **9路、13路、19路**），并自由选择 70 多位明星棋手之一作为你的对手。
    
- **挑战解锁机制**：AI 对手的段位和棋风会自动锁定为该棋手卡牌的专属属性。一旦玩家在实战中战胜对手，**该棋手的卡牌将被永久解锁并加入到玩家的卡牌整备仓库中**。

### 2. 锦标赛模式

- **淘汰赛制**：玩家选择一名初始领袖并携带单卡卡组，报名参加 16人、32人 或 64人 的大型淘汰赛。
    
- **树状晋级表**：游戏会展示一个完全由场景绘制的横向晋级表。当你进行对局时，**后台其他所有的 AI 场次会基于彼此的段位权重进行真实的概率碰撞模拟**。
    
- **战胜夺卡**：当你击败对手后，该对手不仅会被画上红色删除线淘汰，它的卡牌还会**被你永久收服并加入你本届锦标赛的对战卡组中**。随着轮次推进，你的卡组会越战越强。

## 五、 特色辅助系统

- **形势实时判断**：游戏在本地实现了一套高效的“棋子距离影响力衰减算法（Influence Map）”，在不消耗任何 CPU 开销的前提下，在棋盘上实时渲染出精美的半透明红蓝色光影（蓝色为我方控制区，红色为敌方控制区），局势一目了然。
    
- **棋子手顺显示**：提供“隐藏落子顺序”、“只显最后一手”以及“显示全部手顺数字”三种模式，玩家可在对局中一键实时切换。
    
- **多语言国际化**：游戏支持完整的本地化系统。一键切换语言后，不仅本地 UI 会被自动翻译，**发送给大模型的提示词也会动态调整，大模型会自动用你选择的语言（如中文或英文）撰写战术点评**。
    
- **物理加密存盘**：游戏提供完整的断电保存与恢复功能。退出时可选择保存，系统会使用 **AES-256 军工级物理加密** 将棋盘、手牌、用时、提子数、段位和模式安全写入硬盘。再次开局时，系统会智能提示并完美复原战场。

## 六、 运行指南



1. **游戏目录结构**：
    
    codeText
    
    ```
    游戏根目录/
      ├─ M.Y.GO.exe   (Godot 导出的执行程序)
      ├─ M.Y.GO.pck     (Godot 编译的加密资源包)
      ├─ bin/              (外挂式物理 AI 目录)
         ├─ katago.exe     (KataGo 主引擎)
         ├─ katago.cfg     (经过调优的 9x9/13x13/19x19 极轻量配置文件)
         └─ 权重模型.bin.gz (人类模仿神经网络模型)
    ```
    
2. **玩家运行**：  
    玩家只需将压缩包解压，双击运行 M.Y.GO.exe 即可。游戏中的自适应物理寻址代码会自动在游戏同级目录下唤醒 bin 文件夹里的 AI 大脑，无需玩家进行任何复杂的环境配置，体验极佳。
